# Agent Mode Demo

This directory contains a script for demonstrating Copilot's **Agent Mode**.

## Scenario
The file `data_processor.py` contains some placeholder logic.
**Task for Agent**:
"Modify `data_processor.py` to read a CSV file named `input.csv`, calculate the average of the 'score' column, and write the result to `output.txt`. If `input.csv` doesn't exist, create a dummy one with random data first."

## Instructions
1.  Open `data_processor.py`.
2.  Open Copilot Chat (Panel or Inline).
3.  Switch to **Agent Mode** (if applicable, or simply give the high-level instruction).
4.  Paste the task description above.
5.  Observe how the Agent plans, creates the file, writes the code, and executes it.
