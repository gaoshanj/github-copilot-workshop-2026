# GitHub Copilot 深度实战培训 (2026 Edition)

欢迎来到 **GitHub Copilot 深度实战培训** 官方配套资源库。本项目专为 **VS Code 1.109/1.110** 及以上版本设计，旨在帮助开发者从基础到精通，掌握 GitHub Copilot 的最新特性与最佳实践。

## 🎯 课程模块

### 模块一：基础篇 (Basics)
- **核心特性**: Agent Session Management, Copilot Memory, Inline vs Panel Chat
- **位置**: `01-basics/`

### 模块二：进阶篇 (Advanced)
- **核心特性**: Custom Instructions, Prompt Files, Agent Mode vs Plan Mode, MCP Protocol
- **位置**: `02-advanced/`

### 模块三：拓展篇 (CLI & Skills)
- **核心特性**: Copilot CLI, Natural Language to Shell, Custom Skills Development
- **位置**: `03-cli-skills/`

## 🛠️ 环境准备

为了确保最佳的学习体验，请务必满足以下环境要求：

1.  **Visual Studio Code**: 版本 **1.109** 或更高 (建议使用 1.110 Insiders 以获得最新体验)。
2.  **GitHub Copilot Extension**: 确保安装最新版本的 GitHub Copilot 和 GitHub Copilot Chat 插件。
3.  **Node.js**: v18+ (用于 MCP 和 Skills 开发)。
4.  **Git**: 基础版本控制工具。

## 🚀 快速开始

1.  克隆本仓库：
    ```bash
    git clone https://github.com/your-username/github-copilot-workshop-2026.git
    cd github-copilot-workshop-2026
    ```

2.  打开 VS Code：
    ```bash
    code .
    ```

3.  按照各模块目录下的文档进行练习。
